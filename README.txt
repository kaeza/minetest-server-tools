
minetest-tools
--------------

Some sandom tools related to Minetest.

cdn-update.sh
  Script to copy media files for use by the remote media code.

check-mods.py
  Recursively checks a directory to see if all subdirectory are mods for
  Minetest. Also checks for modpacks.

greplog.sh
  Shortcut to grep something on `debug.txt'.

mt_irc.py
  This is a plugin for HexChat and XChat that supports (almost) seamless
  integration of in-game chat (via the IRC mod) and normal chat.
  Install in `~/.config/hexchat/addons', or similar.

server.sh
  Script to restart a server on error. Has some (not so) useful features.
